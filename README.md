# README.md

## 오시랑께 | 전라남도 여행 추천 웹사이트

전라남도 인기 관광지 데이터를 기반으로 사용자 맞춤형 여행지를 추천해주는 웹사이트입니다.
사용자는 여행 테마와 동행 유형을 선택하여 자신에게 적합한 관광지를 추천받을 수 있으며, 관광지 상세 정보와 추천 코스, 여행 일정 저장 기능까지 사용할 수 있습니다.

프로젝트는 HTML, CSS, JavaScript 기반으로 제작되었습니다.

---

# 주요 기능

## 1. 관광지 추천 기능

사용자가 선택한 여행 스타일에 맞는 관광지를 추천합니다.

### 지원 테마

* 휴식 / 힐링
* 맛집 탐방
* 문화 / 예술
* 액티비티

### 동행 유형

* 가족
* 친구
* 커플
* 혼자

선택한 조건을 기반으로 Top 3 관광지를 우선 추천합니다.

---

## 2. 관광지 검색 기능

관광지 이름을 검색하여 원하는 장소를 빠르게 찾을 수 있습니다.

### 기능

* 실시간 검색
* 검색 결과 표시
* 관광지 상세 이동

---

## 3. 관광지 상세 정보 기능

관광지 클릭 시 상세 설명과 추천 코스를 제공합니다.

### 제공 정보

* 관광지 설명
* 관광 유형
* 추천 포인트
* 추천 여행 일정
* 관광지 카테고리

---

## 4. 추천 여행 코스 기능

관광지별 추천 여행 코스를 제공합니다.

### 제공 코스

* 당일치기 코스
* 1박 2일 코스

### 포함 정보

* 시간별 일정
* 이동 추천
* 여행 팁
* 추천 장소

---

## 5. 여행 일정 저장 기능

추천 코스를 사용자의 여행 일정에 저장할 수 있습니다.

### 기능

* 일정 저장
* 일정 삭제
* 일정 목록 확인
* 일정 인쇄

---

## 6. 일정 공유 기능

사용자가 만든 여행 일정을 공유할 수 있습니다.

### 지원 기능

* 링크 복사
* 텍스트 복사
* 인쇄 기능
* 카카오톡 공유 UI

---

## 7. 즐겨찾기 기능

관심 있는 관광지를 저장할 수 있습니다.

### 기능

* 즐겨찾기 추가
* 즐겨찾기 삭제
* 즐겨찾기 목록 표시

---

## 8. 날씨 기반 추천 기능

전라남도 날씨에 따라 추천 관광지를 제공합니다.

### 예시

* 맑은 날 → 야외 관광 추천
* 더운 날 → 실내 관광 추천
* 비 오는 날 → 실내/문화 관광 추천

---

## 9. 데이터 시각화 기능

관광 데이터를 차트 형태로 시각화하여 제공합니다.

### 제공 차트

* 관광지 분류별 분포
* 지역별 관광지 수
* 월별 방문자 추이
* 계절별 인기 지역 분석
* 월별 테마별 관광 분석

### 사용 라이브러리

* Chart.js
